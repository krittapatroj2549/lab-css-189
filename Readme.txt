การเรียนรู้เรื่องภาษา HTML ทำให้ผมได้เข้าใจพื้นฐานการสร้างเว็บไซต์มากขึ้น โดยได้เรียนรู้การใช้แท็กต่าง ๆ เช่น <html>, <head>, <body> 
รวมถึงการจัดรูปแบบข้อความ การเว้นบรรทัด การเว้นวรรค การสร้างลิงก์ และการทำตาราง ซึ่งสามารถนำไปใช้สร้างหน้าเว็บอย่างง่ายได้ด้วยตนเอง ก่อนหน้านี้เคยเรียนผ่านทางยูทูปมาบ้างแล้ว
นอกจากนี้ ผมได้ฝึกแก้ไขข้อผิดพลาดจากการเขียนโค้ด ทำให้มีความรอบคอบมากขึ้น และรู้จักการทดลองปรับโค้ดเพื่อให้ได้ผลลัพธ์ตามที่ต้องการ การเรียน HTML 
ช่วยพัฒนาทักษะด้านการสร้างโครงสร้างพื้นฐานของเว็บไซต์และความคิดเป็นระบบโดยรวมแล้วผมคิดว่าความรู้เรื่อง HTML เป็นพื้นฐานที่สำคัญในการเรียนในรายวิชา CP151 
และสามารถนำไปต่อยอดในการเรียนรู้การเขียนเว็บไซต์และการเขียนโปรแกรมในระดับที่สูงขึ้นต่อไปได้ครับ
